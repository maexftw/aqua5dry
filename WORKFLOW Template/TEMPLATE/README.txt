Ziel
----
Dieses Template liefert dir klare, kurze Anweisungen für deinen Workflow:
Cursor 2.0 → GitHub → Cloudflare Pages (Preview pro PR, Production auf main).
Alle Dateien hier sind reine Textanweisungen – du kannst sie 1:1 ins Repo kopieren.

So verwendest du das Template
-----------------------------
1) Kopiere den Ordner `TEMPLATE/` und den Ordner `.cursor/` in dein Projekt-Repo.
2) Lies `TEMPLATE/WORKFLOW.txt` einmal komplett durch (2 Minuten).
3) Entscheide dich unter `TEMPLATE/DEPLOY-CHOICES.txt` für die Git-Integration (empfohlen).
4) Richte Cloudflare laut `TEMPLATE/CLOUDFLARE-SETUP.txt` ein.
5) Arbeite dann wie in `TEMPLATE/WORKFLOW.txt` beschrieben: Push → Preview prüfen → Merge → Live.

Hinweis
------
Diese Texte sind bewusst knapp gehalten. Sie geben der KI Leitplanken,
ohne sie in ihrer Kreativität einzuschränken.
